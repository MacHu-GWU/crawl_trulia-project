Install
-------------------------------------------------------------------------------

``crawl_trulia`` is released on PyPI, so all you need is:

.. code-block:: console

	$ pip install crawl_trulia

To upgrade to latest version:

.. code-block:: console

	$ pip install --upgrade crawl_trulia
